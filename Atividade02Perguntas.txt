2) A programação java refere ao processo de escrever, compilar e executar codigos feito por essa desenvolvimento de aplicativos
desktop,desenvolvimento web, desenvolvimento de aplicativos moveis, desenvolvimento de sistemas embarcados,entre outros.

3)Uma linguagem de programação Java é uma linguagem de alto nível projetada para ser independente de plataforma e orientada a objetos.
a linguagem Java é conhecida por sua portabilidade, o que significa que os programas escritos em Java podem ser executados em diferentes 
tipos de dispositivos sem a necessidade de modificação do código-fonte.
